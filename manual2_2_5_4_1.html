주요구성
  • 공조기 [1] 수영장 공조기의 주요 역할

     1. 습도 조절
      수영장에서는 증발로 인해 실내 습도가 매우 높아질 수 있습니다.
      공조기는 공기 중 과도한 수분을 제거하여 쾌적한 환경을 유지하고, 곰팡이나 부식을 방지합니다.

     2. 온도 조절   수영장 내부의 공기 온도와 물의 온도를 균형 있게 유지합니다.
      특히, 물 온도와 공기 온도의 적절한 조화는 이용자의 체감 온도를 최적화하는 데 중요합니다.

     3. 공기 정화
      공기 중의 먼지, 미세 입자, 화학물질(예: 염소 증기)을 제거하여 실내 공기를 깨끗하게 유지합니다.
      이는 수영장 이용자의 호흡기 건강을 보호하는 데 중요한 역할을 합니다.

     4. 에너지 회수
      수영장 공조기는 열 회수 장치를 통해 배출되는 공기의 열을 다시 회수하여 새로운 공기를 가열하는 데 사용합니다.
      이를 통해 에너지 소비를 줄이고 난방비를 절감합니다.

     5. 환기 및 공기 순환
      신선한 외부 공기를 공급하고 내부 공기를 순환시켜, 수영장 공기의 질을 유지합니다.
      환기를 통해 냄새와 유해 가스를 배출합니다.

     6. 응축수 관리
      수영장 내부의 습기 응축으로 인해 생기는 물을 효과적으로 배출하거나 재활용하여, 장비와 실내 환경을 보호합니다.
설정방법
  • 유리잔류염소 보정법
  • 초기 화면에서 1번째 버튼을 3회 눌러 calivration 화면을 뛰운다.
  • calivration 화면에서 3번째 버튼을 2회 눌러 DPD-value 값을 뛰운다.
  • 숫자가 깜빡이면 아래방향키를 누르고 1번째 버튼을 눌러 자릿수를 이동하며 윗방향키나 아래방향키를 눌러 측정한 염소 값으로 숫자를 변경한다.
  • 3번째 버튼을 2회 누르고 5번째 버튼을 3회 눌러 초기화면으로 나간 변경이 되었는지 확인한다.




  • 약품 농도 셋팅 설정법
  • 초기 화면에서 1번째 버튼을 7회 눌러 control 화면을 뛰운다.
  • control 화면에서 3번째 버튼을 3회 눌러 set point 값을 뛰운다.
  • 숫자가 깜빡이면 아래방향키를 누르고 1번째 버튼을 눌러 자릿수를 이동하며 윗방향키나 아래방향키를 눌러 수영장의 원하는 염소 농도 값으로 숫자를 변경한다.
  • 3번째 버튼을 3회 누르고 5번째 버튼을 2회 눌러 초초기화면으로 나간 다음 1번째 1회 눌러 변경된 설정치를 확인한다.




  • 유리잔류염소센서 설치 방법
  • 유리잔류염소 센서와 멤브레인 캡을 분리하고 전해액 끝부분을 잘라낸 후 멤브레인 캡 안에 주입한다.
    [공기방울이 생기지 않게 한번에 주입한다]
  • 전해액이 담긴 멤브레인 캡을 센서 본체와 체결한다.
    [일부 전해액과 함께 일부 공기방울이 빠져나오는 것을 확인한다]
  • 고무패킹과 플라스틱링을 그림의 순서와 같이 결합한다.
  • 결합한 염소센서를 하우징 안에 체결한다.
  • 드라이버로 센서 케이블을 결선한다.
    [갈색선:1번 / 희색선:2번 ]
  • 케이블 결선 후 윗 커버를 체결한다.
  • 하우징 밸브를 열어 물이 흐르는 것을 확인한다.




  • 염소 측정기 PH에서 CL 변경법
  • 초기 화면에서 1번째 버튼을 10회 눌러 general setting 화면을 뛰운다.
  • general setting 화면에서 3번째 버튼을 3회 눌러 측정기를 선택창을 뛰운다.
  • 윗방향키나 아래방향키를 눌러 염소 측정으로 변경한다.
  • 5번째 버튼 2번을 눌러 초기 화면에서 측정방법을 변경되었는지 확인한다.
  • 염소 수치 하한선과 상한선을 0.00 ~ 10.00 으로 변경을 한다.
    [숫자 변경은 다른 숫자 변경 방법과 동일일]
